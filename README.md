# Demo application

## installation instructions

1. Clone the repository

```bash
git clone
```

2. (with ddev) Start the ddev environment

```bash
ddev start
```

3. npm install

```bash
npm install
```

4. npm run dev

```bash
npm run dev
```

5. Visit the site

```bash
http://demo.ddev.site
```
